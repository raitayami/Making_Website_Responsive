Grids and spacing

Introduction

Grids are made up of intersecting horizontal and vertical lines

Grid Types

There are various types of grids.

Most common type of grid a web designer uses is the column grid. It breaks up the page into vertical units that span the width of the content

Grid Anatomy

When designing a website, the grid comprises three major components. Columns, gutters and margins.

  1. Columns. Vertical sections that span the width of the page. It is common to see layouts with 12 or 16 columns, but some might only have 3. Defining the number of columns depends on what is appropriate for your design.
  2. Gutter. The space between each column. They help ensure the columns don't run together.
  3. Margins. They appear on the LHS, or RHS of the column-based grid.

Grid Columns

Columns are vertical containers that span the width of the page

Content can span multiple columns. Meaning a website does not need to maintain the same column layout throughout

You are allowed to decide how much the content can span.

No content can spill into the gutter unless it is using the next column

Columns can be used to push, or offset content. Assume you want to display content on the furthest 4 right column while using a 12 column grid. You can offset teh content by 8 columns so the content is pushed to the right 4 columns

Responsive Grid

Many websites utilize responsive design, a set of techniques that allow a websites content to shift based on the device or screen size

Responsive design requires a different number of total columns based on a screen size.

It is a common practice to create a responsive grid size. On a desktop you may have 12 column grid, but a mobile size, you might have a 4 column grid.

Allowing responsive design ensures a seamless/accessible experience for users.

Whitespace

It is important to understand how to use whitespace, as it can enhance designs or hurt it.

Whitespace is just the emptiness between elements

Whitespace doesn't have to be white

Passive Whitespace

Also known as micro white space, it is used to improve the aesthetics of the layout without guiding the user through a specific reading, flow, or content order

Font families have different amounts of passive whitespace, and you can control aspects of them within your design such as line-height or margin

Active Whitespace

Active whitespace is intentional, since passive whitespace occurs more naturally

Also known as macro whitespace, active whitespace refers to enhancing the overall page structure through space to emphasize content or guide users from one point to the next

Adding active whitespace to a site's design, we can spread different sections out, helping the user guide through the page's content more effectively.
